# Fincheck API

## Descrição

Controle de finanças pessoais.

## 🛠️ Setup do projeto

```bash
$ pnpm install
```

## 🚀 Compilando e rodando o projeto

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## ✅ Executando os testes

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
